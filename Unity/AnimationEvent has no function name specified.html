【Unity/エラー解決】
AnimationEvent has no function name specified

  • Unity

    • はじめに

    Unityのお勉強を青和尚本でしていたら，自分のしょうもないミスでAnimationEventでエラーが出たので，忘れないようにメモがわりに解決方法を残しておきます．

    error : AnimationEvent has no function name specified

    症状

    解決方法

    まずはじめに，エラーがおきているアニメーションがつけられているゲームオブジェクトに移動し，Animationウィンドウを開きます．

    下図の赤い丸の中にある白い四角のところを右クリックします．

    Delete Animation EventをクリックしてEventを削除します．

    以下のようになれば完了です．

    原因

    Animationでキーフレームをいじっている時に，Add Event をクリックしてたみたいですね．．．

    まだ自分はAnimation Eventを扱った事がないのですが，なにやら面白そうな機能です．

    Animation Eventを利用すると，アニメーションに合わせてメソッドを使用することができ，アニメーションに合わせて音を出したり，エフェクトを出したり，当たり判定を変えたりすることができるみたいです．

    [Unity] Animation Eventを使いこなそう！
