    概要

    Javaは1995年にサンマイクロシステムズ社から発表された言語です．このJavaはJavaコーヒーが由来でこの印象的な名前も特徴的です． コーヒーの場合はジャバではなく，ジャワと発音するのが一般的です．

    Javaが有名になったのは，Java言語がどんな環境でも動作するように設計されたことでした．Javaは当初オーク(Oak)と呼ばれていましたが，それは デジタル家電向けのコンパクトなプログラミング言語という目的を持っていました．

    Java言語は

    Write Once, Run Anywhere -- １度作成すれば，どこでも実行できる

    という理念のもと設計されています．

    それまでの言語は，Windowsで作ったアプリはWindowsでしか動かなく，UNIXという別のOSで動かそうとするには 異なるOSでプログラムが動けるようにする移植いしょくといった大変な作業が必要でした．

    このようなプログラムのことをクロスプラットフォームやマルチプラットフォームと呼んだりする．
