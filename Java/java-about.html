    概要

    Javaは1995年にサンマイクロシステムズ社から発表された言語です．このJavaはJavaコーヒーが由来でこの印象的な名前も特徴的です． コーヒーの場合はジャバではなく，ジャワと発音するのが一般的です．

    Javaが有名になったのは，Java言語がどんな環境でも動作するように設計されたことでした．Javaは当初オーク(Oak)と呼ばれていましたが，それは デジタル家電向けのコンパクトなプログラミング言語という目的を持っていました．

    Java言語は

    Write Once, Run Anywhere -- １度作成すれば，どこでも実行できる

    という理念のもと設計されています．

    それまでの言語は，Windowsで作ったアプリはWindowsでしか動かなく，UNIXという別のOSで動かそうとするには 異なるOSでプログラムが動けるようにする移植いしょくといった大変な作業が必要でした．

    しかしJava言語を利用することで，移植作業の必要がなくなりました． Javaで作成されたアプリケーションはコンピューターの種類が異なっていても， そのままアプリケーションが実行することができるのです．

    このようなプログラムのことをクロスプラットフォームやマルチプラットフォームと呼びます．

    JVMとは

    JVMとはJava Virtualばーちゃる Machineましーん の略で，Javaの仮想マシンと呼ばれるソフトウェアの働きによってJavaはマシン依存のないプログラムを実現しているのです．

    Javaはソースプログラム（コード）をコンパイルと呼ばれる処理を行うと，バイトコードと呼ばれる特殊な形式のファイルを生成します． バイトコードは機種依存する機械語ではないので，直接実行することはできませんが，バイトコードをJVMで実行させることによって，アプリケーションが 実行されるのです．今は懐かしい，ガラケーでJavaのプログラムが動いていたのも，携帯電話用のJVMが開発し利用されていたからなのです．
