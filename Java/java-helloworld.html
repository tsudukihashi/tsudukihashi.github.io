    実行環境を整えよう

    使用しているOSにあったJDKをインストールしパスを通しましょう．詳しくはGoogleで検索してみてください．
    参考URL : 初心者でも簡単にできる！Javaをインストールする方法(Windows編)

    ちなみにmacならインストールが終わればそのまま使える！macサイコーーーーーーーーーーーーーーー

    さあさ，インストールが終わったら，次はコードをどうやって書くかです！

    すでにお気に入りのエディタがあるひとはそれを使ってください．ない人はvimでもemacsでもいいですが，最初はvisual studio codeをお勧めします．

    ダウンロード

    理由は軽くて，情報が豊富で，カスタマイズしやすいからです．まあぶっちゃけ生産性が上がればなんでも良いです．

    もちろん自分でエディタを作ってもいいですよ！

    コードを書く

    やっと，コードが書けます！嬉しいですね！！！

    HelloWorld.java 
    class HelloWorld{
  public static void main(String[] args){
    System.out.println("Hello, World.");
  }
}

    このコードをHelloWorld.javaという名前で作成して保存してください．Visual Studio Codeでの画面が下図のようになります．

    Visual Studio Codeなら下画面にコマンドプロンプト，ターミナルを表示させることができ，なおかつ開いたときのカレントディレクトリを表示しているため，そこに直接コマンドを打てるので便利です．

    コードの説明は後回しにしてとりあえず実行してみましょう！

    コンパイルをしよう！

    実行するにはjavaは一度コンパイルする必要があります．ターミナルに

    javac HelloWorld.java

    と入力します．すると

    HelloWorld.classというバイナリ形式のファイルが作成されます．

    実行しよう！

    ようやく実行できます．ターミナルに

    java HelloWorld

    と入力してエンターキーを押してください. 今度は拡張子まで打つ必要はありません．

    無事Hello,Worldが表示されました！ 表示された方おめでとうございます．されなかった方は半角で入力しているか，全角スペースを使っていないか，そもそもパスは通っているかどうかなど確認してみてください．

    間違いの中でこそ成長できると思うので，間違えられた方はラッキーですよ！
