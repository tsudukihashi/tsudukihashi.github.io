    コードの内容

    前回書いたコードは以下の通りです．

    HelloWorld.java 
    class HelloWorld
{
    public static void main(String[] args)
    {
      System.out.println("Hello, World.");
    }
}

    クラス

    一行目は classくらす HelloWorldはろーわーるど で始まっています．

    Javaのプログラムのコードは"class"とついたブロックから成り立っています．このブロックのことをクラスと呼び， "class"の次に続く単語がクラス名と呼ばれています．

    Javaのプログラムにはクラスが最低でも一つ必要で，クラス名は自由につけることができます．

    main(めいん)メソッド

    次は publicぱぶりっく staticすたてぃっく voidぼいど mainめいん ( Stringすとりんぐ [ ] argsあーぐす )となっています．

    Javaのプログラムは原則このmain()と書かれた場所から処理が行われます．

    publicはアクセス修飾子と呼ばれるもので詳しくは今後説明してきますが，これがないとエラーになります．

    main()の中にあるものを引数ひきすうと呼びます．
